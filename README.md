# Oinow Aitu App

React приложение для Aitu Apps, предоставляет платформу для создания мультиплеерных игр на 
Aitu Apps.
Бэкэнд хранится [тут](https://github.com/doughnuty/oinow_backend)

## Документация

http://docs.aitu.io/

## Установка

```bash
npm install
npm run start
```
открыть в браузере http://localhost:3000/
